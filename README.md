# Portfolio

A modern personal portfolio built with React and deployed on Vercel.

## Live

https://naveenkrishnanr.vercel.app/

## Features

* Responsive design (mobile + desktop)
* Smooth scrolling and animations
* Timeline for education and experience
* Projects and publications showcase
* Contact section with external links
*Contact.jsx - pulse maker Toggle markers: 1 to turn on, 0 to turn off

## Tech Stack

* React / Vite
* Tailwind CSS
* shadcn/ui
* Vercel (deployment)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
